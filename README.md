# Project Title
SmartStay-Hotel Booking Web Application

## Project Overview
SmartStay is a modern hotel booking web application developed using React. The system allows users to browse hotel rooms, view detailed room information, and book rooms easily. It also includes an owner dashboard where hotel managers can add rooms and manage listings.This project focuses on responsive design, form validation, and dynamic user interaction.

## Features of Project
The SmartStay hotel booking web application allows users to easily browse available hotel rooms and explore detailed information about each room including amenities, pricing, and location. Users can select check-in and check-out dates and book rooms through an interactive booking interface. The system also allows authenticated users to view their bookings and manage their reservations. In addition to user features, the application includes an owner dashboard where hotel owners can add new rooms, view room listings, and manage available accommodations. The application is designed to be responsive and works smoothly across mobile, tablet, and desktop devices, ensuring a user-friendly booking experience.

## Technologies Used
The SmartStay project is developed using modern web development technologies. The frontend of the application is built with React.js, which helps create a dynamic and interactive user interface. React Router is used to manage navigation and routing between different pages of the application. The styling and layout are implemented using Tailwind CSS, which allows fast and responsive UI design. Framer Motion is used for smooth animations and transitions within the interface. User authentication is handled using Clerk Authentication, which provides secure login and user management functionality. The project is built and optimized using Vite, a fast development build tool. Version control and project management are handled using Git and GitHub, and the final application is deployed online using GitHub Pages.
